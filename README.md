# 🍕 Order Management Dashboard

An interactive Excel dashboard designed to analyze sales performance, revenue trends, product demand, and discount impact for a pizzabun business.

---

# 📌 Project Overview

The **Order Management Dashboard** provides a complete overview of business operations through dynamic visualizations and KPI tracking. The dashboard helps monitor:

✅ Total Orders
✅ Revenue Generated
✅ Average Revenue
✅ Discount Analysis
✅ Product Performance
✅ Daily Sales Trends
✅ Agent-wise Performance

The dashboard is fully interactive using **Pivot Tables**, **Pivot Charts**, and **Slicers** for real-time filtering and analysis.

# 🧾 Dataset Information

The dataset contains transactional sales data with the following fields:

| Column Name     | Description             |
| --------------- | ----------------------- |
| Order ID        | Unique order identifier |
| Product ID      | Product code            |
| Order Date      | Date of order           |
| Product Name    | Name of product         |
| Order Type      | Online / Physical Visit |
| Unit Price      | Price per item          |
| Agent           | Sales representative    |
| Quantity        | Quantity sold           |
| Discount %      | Discount applied        |
| Gross Revenue   | Revenue before discount |
| Net Revenue     | Revenue after discount  |
| Discount Amount | Total discount value    |

Dataset used for dashboard creation.

# 🚀 How to Use

1. Open the Excel workbook.
2. Navigate to the **Orders-Dashboard** sheet.
3. Use slicers to filter data by:

   * Order Type
   * Agent
4. Analyze business performance using interactive charts and KPIs.
